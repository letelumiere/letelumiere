## 스쿼드매니아 (iOS/Android) 

- 모바일 게임 피파 온라인의 데이터 정보 제공하는 게임 커뮤니티 앱.
- DB 설계와 개발, 서버 구축, API 개발 등, 백앤드 개발 및 패키지 배포 작업 참여.
